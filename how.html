How Do Neural Networks Function?

Information transfer process, training, and an example.

Most neural networks function by passing information as vectors (mathematical structures) through layers of nodes to reach a resulting output. However, there are many types of neural networks and they have some differences..

The simple structure of neural networks consists of an input layer, a hidden layer, and an output layer. A deeper structure builds on this by having an input layer, several hidden layers, and an output layer. There are also weights between nodes.

Training

To train neural networks, large data sets have to be provided. These data sets can be either unlabeled or labelled. The labelled data sets will train a neural network for classification, while unlabeled data sets will train it for clustering. Classification is the sorting of items into different categories, such as categorizing spam and real email, or recognizing letters from images. Clustering groups similar information together.

Example of Neural Network: Differentiating between squares and circles

(1) The AI will first get input of an image in pixels.

(2) The data of each pixel will be fed into each of the neurons in the first layer.

(3) The first layer is connected to the second layer through channels

(4) Each of these channels have weights

(5) A mathematical formula multiplies the data with the weights and adds a number called the bias

(6) Then the sum is put through an activation function

(7) The activation function determines if the neuron will light up

(8) This way, the data propagates through all the layers

(9) The neuron with the highest value in the output layer will light up

(10) This gives the final result as a probability
